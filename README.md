
  # Community Marketplace Website

**Supabase setup**: Create a `.env` or `.env.local` in the project root and set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`. See `SUPABASE_ENV.md` and `.env.example` for details.

**DB seeds**: If you want admin and example accounts in the DB, follow the instructions in `src/supabase_admin_seed.sql` to create auth users (via the Admin API or Dashboard) and insert matching profile rows.

  This is a code bundle for Community Marketplace Website. The original project is available at https://www.figma.com/design/EjXx4ocqNKZ507Bri5XGxA/Community-Marketplace-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
